# 🚀 Frontend Setup - SmartInvest-App
Bagian ini berisi kode sumber untuk antarmuka pengguna (frontend) yang dibangun menggunakan React.js + Vite.

## Prasyarat
Pastikan kamu sudah menginstal:
- Node.js (versi terbaru sangat disarankan)
- npm (biasanya otomatis terinstal bersama Node.js)

## Cara Menjalankan di Lokal

Masuk ke folder frontend:
```bash
cd fullstack/frontend
```

Instalasi dependensi:
```bash
npm install
```

Jalankan aplikasi:
```bash
npm run dev
```

Aplikasi akan terbuka di browser pada alamat http://localhost:5173
